Firebase Setup
Connect your website to Firebase.
Step-by-Step Guide
  1. Go to console.firebase.google.com
  2. Create Project → name it ahsan-ullah-food-point
  3. Add Web App → Register → copy firebaseConfig
  4. Open Admin/firebase-config.js → paste your config
  5. Realtime Database → Create Database → Start in Test Mode
  6. Authentication → Sign-in Method → Enable Email/Password
  7. Authentication → Users → Add User (your email + password)
  8. Come back here → click Sync Local Data to Cloud
